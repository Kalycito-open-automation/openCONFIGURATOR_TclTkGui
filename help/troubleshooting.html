Trouble Shooting

1. "The file od.xml is missing\n Cannot Proceed\nPlease consult User Manual"

The file od.xml is used for storing the objects for the tool. If the file is missing, new objects cannot be added and, the autogeneration will also fail, so the tool will not allow the user to proceed if the file is missing.

Copy the od.xml from the zip file downloaded from SourceForge into the installed directory and the issue will be fixed.

2. "Tranfer CDC/XAP file"

If the Tranfer CDC/XAP file has any issue, please read through the comments described on top of the Transfer.bat/Transfer.sh file.
